# Real-World Distribution Velocity Network (RDVN)

This project is a hackathon MVP that models real-world consumption velocity across fragmented distribution channels including retail, vending, and institutional environments.

## Core Idea
Capture structured distribution events (sales, restocks, stockouts) and compute velocity metrics to understand how physical distribution structure impacts consumption behavior.

## Stack
- React (dashboard UI)
- Event-based data model (JSON schema)
- Optional Solana integration for event anchoring

## Optional Blockchain Layer
Experimental integration with Solana for timestamping and anchoring selected real-world distribution events.

## Status
Early-stage hackathon MVP / concept prototype
